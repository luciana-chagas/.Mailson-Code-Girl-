O que é programação html?

A linguagem usada para criar um website é chamada de HTML, com ela é possível escrever um código para instruir o navegador como mostrar uma página web. Apesar da expressão Programação html ser muito comum, o HTML não é considerado uma linguagem de programação e sim uma linguagem de marcação. HTML é uma abreviação de Hyper Text Markup Language, ou em português, Linguagem de Marcação de Hipertexto. Assim, na verdade quando falamos em programação nos referimos ao conjunto de regras, códigos, ou grupo de instruções que são processados pelo navegador. A linguagem html permite criar a estrutura de um website e apresentar seu contéudo usando tags.

O que são tags?

“Tag” em inglês quer dizer etiqueta. As tags na internet são palavras que servem justamente como uma etiqueta e ajudam na hora de organizar informações, agrupando aquelas que receberam a mesma marcação, facilitando encontrar outras relacionadas. Em linguagem HTML, podemos colocar dentro de cada etiqueta (tag) inicial uma caracterísitca específica. Chamamos isso de Atributo.

Existem diversas formas de classificar e organizar informações. A mais conhecida é por hierarquias, como se fossem colocadas pastas mais específicas dentro de pastas mais genéricas. Essa organização hierárquica é a forma mais utilizada para armazenar e encontrar páginas na internet, por exemplo.

Com já disssemos, o HyperText Markup Language, ou HTML, é utilizado para definir o local de cada item em seu navegador. Através dele você consegue posicionar o cabeçalho, os menus, os botões, entre outros detalhes, contudo para colorir, organizar e dar aqule toque personalizado para sua página web, contamos com uma outra linguagem de nome css (o Cascading Style Sheets), que compõe a aparência, é ele que determina todo o layout, oferecendo diferentes cores, mostrando todos os backgrounds que você escolheu e exibindo animações e efeitos em 3D.).
